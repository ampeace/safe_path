# Safe Path - Smart Navigation System

A web-based application that helps drivers find the safest route by avoiding dangerous areas, accidents, and bad weather.

## Features
- **Smart Route Planning**: Automatically calculates the safest path to your destination.
- **Real-time Updates**: Monitors traffic, accidents, and weather conditions.
- **User Authentication**: Secure sign-up and login system.
- **Modern UI**: Clean, intuitive interface with theme support.

## Installation

### Prerequisites
- Node.js and npm
- Python and pip
- MongoDB or other compatible database

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the server:
   ```bash
   python main.py
   ```

### Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Usage
1. Open your browser and go to `http://localhost:3000`
2. Sign up or log in to your account
3. Enter your destination and let Safe Path find the safest route for you
